train.evaluate_model(best_val_metric, best_model, test_set, criterion, device, type_model)[source]

This function evaluates the best model on a test dataset.

Parameters:

  • best_val_metric (dict) – The best validation metric obtained during training.

  • best_model (torch.nn.Module) – The best model obtained during training.

  • test_set (torch.utils.data.DataLoader or test_svm) – Dataloader or SVM dataset based on the model

  • criterion (torch.nn.modules.loss._Loss or Funcion) – The criterion to use for calculating loss during evaluation.

  • device (str) – The device on which to evaluate the model (e.g., ‘cpu’, ‘cuda’).

  • type_model (str) – The type of the model (e.g., ‘ResNet’, ‘AlexNet’).

Returns:

Returns the metrics and confusion matrix for the test dataset.

Return type:

tuple (list, numpy.ndarray)
